Class Rs.ext.form.AutoCompleteField

Package:Rs.ext.form
Defined In:rs-ext-debug.js
Class:AutoCompleteField
Extends:Object
自动完成文本框(简易版望远镜和文本框的合体),不支持多选,默认是显示10条数据,没有翻页,通过用户输入信息,进行更加精确查询
有两种模式:严格模式,类似望远镜的方式;模糊模式:类似建议文本框方式.模式是建议文本框
两种模式的切换,请查看setStrictModel方法
