Class Rs.ext.form.PeriodField

Package:Rs.ext.form
Defined In:rs-ext-debug.js
Class:PeriodField
Extends:Object
核算期控件，用于编辑和显示核算期.
例如： 2010/05 表示2010年05核算期
一般情况核算期由年和月份构成.
