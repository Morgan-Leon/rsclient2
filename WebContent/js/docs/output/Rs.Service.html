Class Rs.Service

Package:Rs
Defined In:rs-debug.js
Class:Service
Extends:Object
A static Rs.data.Service instance that can be used to call services. See Rs.data.Service for supported methods. 
Rs.Service.call({
	url : 'service.jsp',
	method : 'getJSON',
	accept : 'json',
	params : { id : 100},
	success : function(sheet) {
		Rs.fly('out').update(sheet);
	}
});


This class is a singleton and cannot be created directly.

This class has no public properties.
